진행 과정

1. 작업 세분화

  • # 메인페이지
  • # 서브페이지1 ( ABOUT US )
  • # 서브페이지2 ( 제휴문의 + 헤더 + 푸터 )
  • # 서브페이지3 ( 도시락멤버스 )

팀프로젝트를 처음으로 작업하게 되어 소통 방법을 찾는 것 부터 시작해서 각자 작업한 작업물을 하나로 합치는 방법 등 팀으로 작업하는 법에 대해 직접 부딪혀 가며 배울 수 있었습니다.

2. 공유 및 취합

동일하게 적용되는 부분들은 공유해서 사용하고 각자 작업한 작업물을 하나로 취합

3. github에 푸쉬

하나로 합쳐진 작업물을 github에 푸쉬해 배포

문제 & 해결

  • header

    메뉴에 마우스를 올렸을 때 밑줄이 왼쪽에서부터 천천히 길게 등장해야 했는데 transition으로도 원하는 움직임이 나오지 않아 ::after 로 left 값을 0에서 20으로 자연스럽게 움직이도록 했습니다.

  • 작업물 취합 과정

    각자 작업한 작업물들을 하나로 취합하는데 각자 사용한 태그나 클래스명, 아이디명이 달라서 생각했던 것과 다르게 한번에 완성되지 않았고 작업된 작업물을 해치지 않으면서 합치는 방법에 대해 팀원과 의논했습니다.

    각 작업물을 크게 감싸는 태그를 만들고 헷갈리지 않도록 클래스명을 통일시켜 문제를 해결했습니다.
